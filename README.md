# Brooke & Belle — Design Files

All 28 shirt designs as SVG and PNG files, with a visual HTML mapping to Shopify listings.

## What's here

- **`all-designs-svgs.zip`** — Download this. Contains all 28 designs as both `.svg` and `.png`, plus the HTML mapping file.
- **`design-mapping.html`** — Open this in a browser to see every design with its name, reveal phrase, filenames, and Shopify product links.

## Collections

| Collection | Designs | Examples |
|-----------|---------|---------|
| Acronym Shirts — Batch 1 | 10 | PTO, RSVP, VIP, CEO |
| Acronym Shirts — Batch 2 | 16 | STFU, LMAO, GTFO, BRB |
| Maxxing Collection | 2 | MARGARITAMAXXING, RETIREMAXXING |

## File naming

Every file is named `{acronym}-design.svg` / `{acronym}-design.png`. For example:
- `vip-design.svg` → VIP: Very Into Pinot
- `margaritamaxxing-design.svg` → MARGARITAMAXXING

## How to use

1. Download `all-designs-svgs.zip`
2. Unzip it
3. Open `design-mapping.html` in your browser to see which file goes with which Shopify listing
